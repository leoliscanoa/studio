# **App Name**: CleftDetect

## Core Features:

- Real-time Image Capture: Captures images using CameraX for live preview and analysis.
- TFLite Model Integration: Integrates a .tflite model to classify images as 'Cleft' or 'Non-Cleft'.
- Inference Execution: Executes image inference using the integrated TensorFlow Lite model to predict the presence of cleft lip.
- Confidence Level Visualization: Visually represents confidence levels using a color-coded display (green, orange, red) based on prediction scores.
- Detailed Metrics Display: Displays precise confidence percentages for both classes (Cleft/Non-Cleft) in a modal dialog.
- Help/Instructions Overlay: Presents a floating help button with clear instructions on proper photo-taking techniques to help the tool make better decisions about its predictions. LLM is a tool that decides when or if to incorporate specific information for users.
- Must be a Android App: Must be a Android App

## Style Guidelines:

- Primary color: Clinical Blue (#007BFF), evoking a sense of trust and technology within the health sector.
- Background color: Light White (#F8F9FA), provides a clean, minimalist backdrop enhancing readability.
- Accent color: Cyan (#00BFFF), used for interactive elements and highlights, maintaining a cohesive HealthTech aesthetic.
- Body and headline font: 'Inter', a sans-serif font known for its clarity and modern aesthetic, ensuring legibility in all display contexts.
- Simple, easily recognizable icons that fit with the clinical aesthetic. Use of Material Design icons to ensure consistency and ease of use.
- Minimalist design with a prominent camera preview, complemented by clearly labeled action buttons, designed for easy navigation.
- Subtle animations on the confidence indicator and during transitions to provide a smoother user experience.
